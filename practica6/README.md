# Practica 6 - Keylogger

## Descripción

Simulación educativa del funcionamiento general de un keylogger desarrollado para la materia de Criptografía y Seguridad 2026-2.

---

## Requisitos

- Python 3
- Linux / Unix
- yagmail
- pynput
- smtlib 
- keyboard
- pyxhook


---

## Archivos

- keylogger.py
- Cleanup.sh
- output.txt
- README.md

---

## Ejecución

Dar permisos:
chmod +x Cleanup.sh

Ejecución del keylogger:
python3 keylogger.py

Eliminar el rastro: 
./Cleanup.sh
